## 📥 Installation & Local Setup

1️⃣ Clone the repository:
```bash
git clone https://github.com/your-username/veda-core.git
```

2️⃣ Navigate to the project directory:
```bash
cd veda-core
```

3️⃣ Install dependencies:
```bash
npm install
```

4️⃣ Start the development server:
```bash
npm start
```
